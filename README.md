# DataX-Labs-Task-3

## Sales Dashboard using Power BI

This project presents an interactive sales dashboard built using Microsoft Power BI and a Kaggle sales dataset.

### Features

* KPI cards showing Total Sales, Total Orders, and Average Sales
* Sales by Product Line
* Sales by Country
* Monthly Sales Trend
* Sales by Deal Size
* Interactive slicers for Country and City

### Tools Used

* Microsoft Power BI
* Kaggle Sales Dataset

This dashboard provides interactive insights into sales performance across different products and locations.
